# wuchat-front

这是网页聊天室的前端代码，后端见 [wuchat-backend](https://github.com/liucxi04/wuchat-backend)。

本项目参考了 [vue-min-chat](https://github.com/WeiLin-Liao/vue-min-chat)。

主要使用的技术栈如下：
 - vue.js
 - Element UI 
 - node.js
 - websocket
